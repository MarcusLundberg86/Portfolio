SOAPP

Social sustainability

    Background

    This project was done to investigate how we can create sustainable design for a successful integration.

    The problem:

    Alienation, Information availability, insecurity, cultural differences, lack of communication between authorities.

    Our concept

    The solution we came up with was SOAPP, a smartphone application with the purpose to reduce the processing time in the immigration process. This was done by integrating immigrants into Swedish society through education, challenges, activities and tests. The application was in first hand presented in the user’s native language. SOAPP uses gamification to activate and motivate the user to continue playing and learn about Sweden. This would increase the user’s knowledge about their unknown new country of residence and give them better tools to interact with swedes and the society. The project is grounded in sustainability with integration as a theme with data from interviews, observations and lectures from the Red cross and Länsstyrelsen. Grounded in goal oriented design process with contributions from sustainability frameworks like SROI (Social Return on Investment) and “Social Value Mapping”.

  • My Role: Concept, UX

  • Background

    •

    This project was conducted in cooperation with HPI (Health Profile Institute) which provides organisations with services to let them know about their employees’ health and wellbeing. The service in this case was HPB (Hälsoprofilsbedöming/Health Profile Evaluation) which is a service used by most organisations who are interested in their employees’ health status. Organizations can see if they are effected financially by employee’s health status and which actions that can be taken to improve their health.

  • Mission

    •

    Our mission was to present a solution for an easier way to communicate the statistics gathered during a HPB.

  • Our concept

    •

    By creating a new platform for HPB we saw new areas of use. Instead of only giving organisations and personnel statistics and numbers, we created a way for both parties to keep more up to date about their and the organisations health. With this new platform HPI can deliver value to customers and create an action plan for increased health and make it possible for the user by themselves update their health profile with data. This way the organization get a more up to date depiction of the employees’ health. This service will make it more attractive for organizations to commit to a longer contract with HPI and will create a stable income for HPI.

Manomotion

Usability evaluation (VR)

  • Background

    •

    ManoDraw is a MobileVR drawing application that enables smartphone users to interact in a virtual reality drawing scenario using hand gestures. The application is showcasing the core 3D Gesture Technology developed by the R&D team at ManoMotion. The gesture analysis technology developed by ManoMotion is fully software-based and does not require extra hardware or sensors. The smartphone’s camera is the main sensor used to capture and analyze hand movements in front of the device. ManoDraw provides different hand gestures in 3D space for opening/closing menus, selecting colors, and 3D drawing in virtual reality.

  • Mission

    •

    Evaluate and investigate the existing features of the current application in terms of usability of the offered gestures, design of the interface and overall user experience. The work will involve getting first hand experience as a user of the application, interacting with other users, studying how users interact with it and how they experience it. The goal is to create a concept with suggestions for improvements and provide a clear argument for why this concept would be better.
